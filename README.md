# Todo Tracker

Todo Tracker — простое Android-приложение для работы с tasks todo list.

Приложение позволяет просматривать задачи, искать их по названию, открывать подробную информацию о задаче и менять её статус: выполнена / не выполнена. Также приложение поддерживает переключение между светлой и тёмной темой.

## Реализованный функционал

### Просмотр списка задач

На главном экране отображается список todo-задач.

Реализовано:

- loading state при загрузке данных;
- data state при наличии задач;
- no data state, если задач нет или поиск ничего не нашёл;
- search bar для фильтрации задач по названию;
- переход на детальный экран задачи по нажатию на элемент списка;
- переход на экран настроек.

### Экран задачи

На экране задачи отображается подробная информация о выбранной todo-задаче.

Реализовано:

- отображение названия задачи;
- отображение id задачи;
- отображение текущего статуса задачи;
- возможность отметить задачу как выполненную;
- возможность вернуть задачу в состояние невыполненной;
- кнопка возврата назад.

### Настройки

На экране настроек реализована смена темы приложения.

Реализовано:

- переключение между светлой и тёмной темой;
- применение выбранной темы ко всему приложению;
- кнопка возврата назад.
